# 🚀 Project CLI

مشروع CLI متكامل مع **Gemini AI** والرسوم البيانية، مبني على Node.js و Express.

## ✨ المميزات

- **🤖 ذكاء اصطناعي متكامل** مع Gemini AI
- **📊 رسوم بيانية ذكية** تولد تلقائياً من البيانات
- **🔐 نظام مصادقة آمن** مع JWT
- **👥 إدارة مستخدمين متكاملة**
- **📈 تحليلات وأدوات بيانات متقدمة**
- **⚡ أداء عالي** مع تخزين مؤقت وضغط
- **🔒 أمان متقدم** مع تحقق من الهجمات
- **📱 واجهة ويب تفاعلية**

## 🛠️ التثبيت

### المتطلبات الأساسية

- Node.js 18+
- MongoDB 5+
- npm 8+
